# Xiaoxiao's Blog

This is a simple but powerful blog, I try to build the blog more beautiful, more convenient. 

## Server Requirements

- PHP >= 7.0.0
- Node >= 6.x
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
- Redis PHP Extension
